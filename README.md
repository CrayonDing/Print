# Print
一个由Java编写的简单绘图程序。自己的课程设计，顺手开源，欢迎大佬指点。

By：CrayonDing(CrayonDing@gmail.com)

------

### 功能

* 画笔、橡皮功能
* 绘制直线、矩形、圆形、椭圆、圆弧、三角形、五边形、六边形
* 可选择图形是否填充、以及边框线宽
* 插入自定义文字
* 图片打开与绘图保存

------
